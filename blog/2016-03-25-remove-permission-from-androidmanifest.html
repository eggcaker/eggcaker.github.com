Remove permission from Androidmanifest.xml

In the android development, you probably has tons of dependencies, sometime if you using a library which add some unwanted permissions and you cannot change the upstream coode and don't want to fork it. You can just remove that permission with simple configuration in the AndroidManifest.xml file 

<uses-permission android:name="android.permission.CALL_PHONE" tools:node="remove" />

happy coding...
